Feedstock for [sherpa](https://gitlab.com/sherpa-team/sherpa) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Sherpa: multi-purpose Monte Carlo event generator for particle physics

## Architectures

| Architecture | Latest published |
|--------------|------------------|
| linux-amd64 (`linux-64`) | ✅ `2.2.16` |
| linux-arm64 (`linux-aarch64`) | ✅ `2.2.16` |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge sherpa
```
